    * Selectors
        * Basic selectors (`#,.,<tag_name>,*`)

```css
            /* Element selector */
            body{
                background: red;
            }

            /* Id selector */
            #header{
                background: #ddd;
            }

            /* Class selector */
            .github{
                color: #000;
                text-decoration: none;
                background: blue;
            }

            /* Universal selector */
            *{
                font-family: Helvetica;
            }
```
